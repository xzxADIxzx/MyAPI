Crypt
Crypt
def crypt(msg, key, ED)
str msg - Text for crypting
int msg - Encryption key
bool ED - True : Encrypt, False : Decrypt

Completely encrypts the text using a set of functions.
crypt("Hello World!", 123456, True)
>>> "208487147732414421680879701919434547470269630939160523743"
crypt("208487147732414421680879701919434547470269630939160523743", 123456, False)
>>> "Hello World!"

def translate(msg, key, ED)
str msg - Text for crypting
int msg - Encryption key
bool ED - True : Encrypt, False : Decrypt

Translates text from characters to numbers and vice versa.
translate("Hello World!", 123456, True)
>>> "114109123123129300144129135123107301"
translate("114109123123129300144129135123107301", 123456, False)
>>> "Hello World!"
