Crypt
def crypt(msg, key, ED)
str msg - Text for crypting
int msg - Encryption key
bool ED - True : Encrypt, False : Decrypt

Completely encrypts the text using a set of functions.
crypt("Hello World!", 123456, True)
>>> "208487147732414421680879701919434547470269630939160523743"
crypt("208487147732414421680879701919434547470269630939160523743", 123456, False)
>>> "Hello World!"

Plus Msg
def plus_msg(msg, key, ED)
str msg - Text for crypting
int msg - Encryption key
bool ED - True : Encrypt, False : Decrypt

Divides the message into three-digit chunks.
"000111222333444555666777888999"
"000", "111", "222", "333", "444", "555", "666", "777", "888", "999"
Takes two pieces and does what is indicated in the example below.
1 - msg[first] += msg[second] 1 - "111", "111", "555", "333"
2 - msg[second] += msg[first] 2 - "111", "222", "555", "888"
3 - msg[second] += key0 3 - "111", "223", "555", "889"
4 - msg[first] += key1 4 - "134", "223", "578", "889"
int key0 - first digit from key
int key1 - first two digits from key


Plus Key
def plus_key(msg, key, ED)
str msg - Text for crypting
int msg - Encryption key
bool ED - True : Encrypt, False : Decrypt

Divides the message and the key into three-digit chunks. And adds to each part of the message all parts of the key. If a part of the message is more than 3 characters, then 1000 is subtracted from it.
for y in range(0, len(key)):
msg[x] += key[y]
while msg[x] > 999:
msg[x] -= 1000



Plus Power
def plus_power(msg, key, ED)
str msg - Text for crypting
int msg - Encryption key
bool ED - True : Encrypt, False : Decrypt

Divides the message into three-digit chunks. And adds a (third key number) 2 to each part of the message.

Move
def move(msg, key, ED)
str msg - Text for crypting
int msg - Encryption key
bool ED - True : Encrypt, False : Decrypt

Each n character is shifted right to the next n character. n = sixth number of the key.
move("000111222333444555666777888999", 246135, True)
>>> "800110222313444355665777868999"
move("800110222313444355665777868999", 246135, False)
>>> "000111222333444555666777888999"

Dict Inv
def dict_inv(dictionary)
dict dictionary - Dictionary to invert

Inverts the dictionary. key = value, value = key. Used for decryption in the translate function.
translation = { "A": "100", "a": "101", "B": "102", "b": "103", "C": "104", "c": "105", "D": "106", "d": "107", "E": "108", "e": "109"}
dict_inv(translation)
>>> { "100": "A", "101": "a", "102": "B", "103": "b", "104": "C", "105": "c", "106": "D", "107": "d", "108": "E", "109": "e"}
