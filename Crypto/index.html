Crypt
def crypt(msg, key, ED)
str msg - Text for crypting
int msg - Encryption key
bool ED - True : Encrypt, False : Decrypt

Completely encrypts the text using a set of functions.
crypt("Hello World!", 123456, True)
>>> "208487147732414421680879701919434547470269630939160523743"
crypt("208487147732414421680879701919434547470269630939160523743", 123456, False)
>>> "Hello World!"



Translate
def translate(msg, key, ED)
str msg - Text for crypting
int msg - Encryption key
bool ED - True : Encrypt, False : Decrypt

Translates text from characters to numbers and vice versa. Uses a dictionary for translation.
translate("Hello World!", 123456, True)
>>> "114109123123129300144129135123107301"
translate("114109123123129300144129135123107301", 123456, False)
>>> "Hello World!"



Plus Msg
def plus_msg(msg, key, ED)
str msg - Text for crypting
int msg - Encryption key
bool ED - True : Encrypt, False : Decrypt

Divides the message into three-digit chunks.
"000111222333444555666777888999"
"000", "111", "222", "333", "444", "555", "666", "777", "888", "999"
Takes two pieces and does what is indicated in the example below.
msg[first] += msg[second]
msg[second] += msg[first]
msg[second] += key0
msg[first] += key1
int key0 - first digit from key
int key1 - first two digits from key


Plus Key
def plus_key(msg, key, ED)
str msg - Text for crypting
int msg - Encryption key
bool ED - True : Encrypt, False : Decrypt

Divides the message and the key into three-digit chunks. And adds to each part of the message all parts of the key. If a part of the message is more than 3 characters, then 1000 is subtracted from it.
for y in range(0, len(key)):
msg[x] += key[y]
while msg[x] > 999:
msg[x] -= 1000
